## Generating types

To generate the types you have to create an access token on you distant supabase project

`npx supabase login`

Provide the Access Token

Then create a lib directory

Then run

`npx supabase gen types typescript --project-id abc123 > lib/database.types.ts `

Where abc13 is your project ID on supabase
